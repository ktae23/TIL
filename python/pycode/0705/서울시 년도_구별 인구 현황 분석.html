※ 참고사항 : axis = 0 vs axis = 1¶

-. axis = 0은 dataframe 행 단위를 수정할 때 필요한 파라미터 값이다. -. axis = 1은 dataframe 열 단위를 수정할 때 필요한 파라미터 값이다. -. -. drop( ) 함수에 index, column이라는 파라미터를 사용하지 않는다면 axis=0 또는 axis=1 파라미터값을 넣어줘야 한다.
